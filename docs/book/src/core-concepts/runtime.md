# 运行时

> **Note / 注意**: This chapter will be available starting Phase 1.
>
> **注意**：本章将在第1阶段开始时可用。

The Nexus runtime provides:
- Thread-per-core task scheduling
- io-uring based I/O operations
- Ownership-based buffer management

Nexus 运行时提供：
- Thread-per-core 任务调度
- 基于 io-uring 的 I/O 操作
- 基于所有权的缓冲区管理
