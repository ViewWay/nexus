# Spring AOP API

上一章描述了 Spring 对 @AspectJ 和基于 schema 的切面定义提供的 AOP 支持。本章将讨论较低级别的 Spring AOP API。对于常见应用，我们推荐使用上一章中描述的、结合 AspectJ pointcuts 的 Spring AOP。

## 章节摘要

- Spring 中的 Pointcut API
- Spring 中的 Advice API
- Spring 中的 Advisor API
- 使用 `ProxyFactoryBean` 创建 AOP 代理
- 简洁的代理定义
- 使用 `ProxyFactory` 编程创建 AOP 代理
- 操作被通知对象
- 使用"自动代理"功能
- 使用 `TargetSource` 实现
- 定义新的 Advice 类型
